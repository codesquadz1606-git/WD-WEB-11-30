COntrol-Flow Statement : Loop
Iteration on a condition until it turns out to be false.
3 types of loops :
i. for
    syntax
    for(start;stop;step){code}
    
ii. while
    syntax
    start;
    while(stop){
        code
        step;
    }

iii. do-while
    syntax
    start;
    do{
        code
        step;
    }while(condition);
